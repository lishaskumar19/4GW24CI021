Concept: Process creation

Uses system calls like fork()

Demonstrates parent and child processes

# os2.c

Concept: Process execution

Uses exec() family functions

Replaces process memory with a new program

# os3.c

Concept: Process synchronization

Demonstrates coordination between processes

# os4.c

Concept: CPU Scheduling

# os5.c

Concept: Memory Management

# os6.c

Demonstrates deadlock situations or prevention

# os7.c

Uses file system calls like open, read, write

# os9.c
